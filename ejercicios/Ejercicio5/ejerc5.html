Descripción del Ejercicio 5

A partir de los documentos HTML que se proporcionan, debes escribir las reglas CSS necesarias para lograr 2 maquetados web que tengan el mismo aspecto (o lo más cercano posible) que la imagen adjunta.El maquetado debe realizarse 2 veces, con técnicas diferentes: Fijas y Fluidas. El CSS se escribe en cada HTML mediante un bloque style
